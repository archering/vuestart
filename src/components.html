组件component 类似于angular里面孤立的作用域的directive ，全局的组件或是本地的组件都是孤立作用域的，都有自己的data， global component 和 local component 都是孤立作用域的； Vue 实例类似 scope ，component 要使用new Vue() 定义的data或者方法或者computed ，需要通过一个特殊变量数组props=[] .

global-compoennt 和 local-compoent 不同有两处。 1， 定义的地方不同， global-component 通过Vue的静态方法Vue.component() 定义 local-component 通过 Vue的初始化参数 new Vue({ components :{ } }) 来定义 2, 使用的地方不同， global-component 可以mount 到任意的节点， local-component只能mount到定义它的节点
